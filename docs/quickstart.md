# Quickstart

Dummynator quickstart

# Preliminaries

## Imports


```python
import dummynator
import numpy as np
import pandas as pd

import sklearn
from sklearn.datasets import load_iris
```

# Setup


```python
iris = load_iris()

X = iris.get('data')
y = iris.get('target')

matrix = np.c_[X, y]
matrix.shape
```

# Fit


```python
from dummynator import Dummynator
```


```python
clf = Dummynator()
```


```python
clf.fit(matrix, strategy='prior')
```
